# Drapeaux 🌍

Collection de drapeaux réalisés en JavaScript dans le cadre du cours de programmation 3D.

Même si ces drapeaux sont en 2D, ils ont été développés avec les notions vues en cours : rendu graphique, animation, interaction utilisateur et manipulation du canvas.

Certains sont des devoirs dans le cadre du cours, d'autres des drapeaux faits dans le cadres de mes révisions avant examen.

---

## 📚 Contenu du projet

Ce dépôt regroupe différents drapeaux programmés individuellement, chacun servant à pratiquer des concepts graphiques et interactifs.

Exemples de fonctionnalités présentes :

- Dessin de formes géométriques
- Gestion des couleurs et proportions
- Animations
- Interactions souris / clavier
- Drag and drop
- Transformations graphiques

### 🇬🇧 Angleterre

## 🎯 Objectif pédagogique

Le but de ces exercices était de développer :

- la logique de programmation graphique ;
- la compréhension des coordonnées et transformations ;
- l’animation en temps réel ;
- les interactions utilisateur ;

## ▶️ Exécution

1. Cloner le dépôt :

```bash
git clone https://github.com/MagalieAbada/Drapeaux.git
```

2. Ouvrir le projet dans VS Code ou un autre éditeur.

3. Lancer avec une extension comme **Live Server** ou ouvrir les fichiers HTML dans un navigateur.
